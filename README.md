* 두 수의 합
* 배열의 평균값

* 가운데 글자 가져오기

* 핸드폰 번호 가리기
* 없는 숫자 더하기
* 자연수 뒤집어 배열로 만들기
* 문자열 내 p와 y의 개수
* 자릿수 더하기

# 1. 배열 / 리스트

* 같은 숫자는 싫어

* 예산
* 체육복
* 모의고사
* 행렬의 덧셈
* 최대공약수와 최소공배수

* 푸드 파이트 대회

# 2. 스택

* 크레인 인형뽑기 게임

* 괄호 회전하기

# 3. 큐 / 덱

* 프린터

* 베스트앨범

# 5. 정렬

# 8. DFS / BFS

# 11. 구현 / 시뮬레이션
